WEB SERVICE

企画・情報設計からデザイン、HTMLコーディング、アニメーションによる演出まで内制作業でシームレスに一気に行います。
全てを内制作業で行うので、最初から最後までお客様の納得のいく品質までしっかりと丁寧に仕上げられることが弊社の強みになります。
撮影やコピーライティング、大規模なシステム開発等が必要な場合も、体制のある専門のパートナー会社と連携して行いますで、
お客様からは安心して全てお任せ頂けるとのお声を頂いています。

GRAPHIC SERVICE

企画、コピーライティング、ロゴやグラフィックツールのデザインまでグラフィックデザインのパートナー会社とウェブと連携して行います。
ウェブの制作を進めながら弊社がアートディレクションを行うことで、ウェブとグラフィックの制作をシームレスに並行して制作できることが弊社の強みになります。
特にショップオーナーの方からはウェブとグラフィックの世界観を統一できることや、複数の制作会社に依頼する手間も省けて有難いとのお声を頂いています。

case study

case 01

グラフィックデザイン関係の方

グラフィックツールの世界観に合わせたウェブサイトや、コーディングのみのご依頼も承ります。企画段階からも参加できますので、まずはお気軽にご相談ください。

実績例：SHIROKURO. / RED HAT FORUM TOKYO 2017 / HITSFAMILY

case 02

ショップオーナーの方

ショップ立ち上げ時に必要なウェブサイトや、名刺、ショップカード、カタログ等のご依頼を承ります。又、内装につきましてもパートナー会社をご紹介できますのでお気軽にご相談ください。

実績例：Natupa / bulan / 1one

case 03

会社の代表・ご担当者の方

コーポレートサイトの立ち上げ・リニューアル・ランディングページ、名刺、パンフレット等のご依頼承ります。大規模なシステム開発が必要な場合も、パートナー企業と連携して行えますのでお気軽にご相談ください。

実績例：peak 1 / KAJIKENSEIJISHA / LINE-INC.

case 03

作家・デザイナーの方

ポートフォリオサイトやブランドサイトなど承ります。世界観を重視して、シンプルなものからエッジを効かせた個性的なデザインまで幅広くご提案致しますのでお気軽にご相談ください。

実績例：DAISY BALLOON / NOZOMI WATANABE / Suwa Design Studio

case 05

ネットビジネスをお考えの方

ウェブマガジン、ウェブサービス等のネットビジネスのご依頼も承ります。大規模なシステム開発が必要な場合も、パートナー企業と連携して行えますのでお気軽にご相談ください。

実績例：deltaphoto / HITSPAPER / YKK AP
